#  Linux DevOps Journey

Este repositório documenta minha jornada de aprendizado em Linux com foco em DevOps.

A ideia é evoluir diariamente, praticando comandos, criando scripts e desenvolvendo automações que simulam cenários reais de um ambiente de produção.

---

##  Objetivo

Desenvolver habilidades práticas em:

- Linux (administração de sistemas)
- Bash scripting
- Automação
- Monitoramento de recursos
- Fundamentos de DevOps

---
##  Estrutura do Projeto 

O repositório está organizado em módulos diários de aprendizado:

linux-devops-journey/
├── day01/
├── day02/
├── day03/
├── ...


Cada pasta representa um dia de estudo prático e contém:

- Scripts desenvolvidos
- Anotações e aprendizados
- Um README próprio com explicações detalhadas

A ideia é manter o projeto escalável.

---

##  Tecnologias utilizadas

- Linux (Ubuntu)
- Bash
- Git e GitHub

---

##  Aprendizados até agora

- Criação e execução de scripts (`.sh`)
- Uso de variáveis no Bash
- Estruturas condicionais (`if`, `else`)
- Pipes (`|`) e manipulação de saída
- Redirecionamento (`>`, `>>`)
- Permissões de arquivos no Linux
- Noções de monitoramento de sistema

---

##  Próximos passos

- Automação com `cron`
- Monitoramento contínuo
- Logs e alertas mais avançados
- Introdução a ferramentas DevOps reais

# Linux DevOps Journey

Este repositório documenta minha jornada prática em Linux, Shell Script e automação, com foco no desenvolvimento de habilidades para atuar em DevOps.

## 📁 Estrutura do Projeto

linux-devops-journey/
└── day01/
    └── desafio.sh

## 🚀 Day 01 - System Info Script

Script em Bash para coleta de informações do sistema:

- Usuário atual
- Nome da máquina (hostname)
- Diretório atual
- Arquivos do diretório
- Data e hora

## 🛠️ Tecnologias utilizadas

- Linux (Ubuntu)
- Bash
- Git e GitHub

## 🎯 Objetivo

Desenvolver habilidades práticas em:

- Comandos Linux
- Automação com Shell Script
- Versionamento com Git
- Base para atuação em DevOps

## 📈 Evolução

Este repositório será atualizado continuamente com novos desafios e scripts, simulando simulando situações reais de automação e infraestrutura.
